# Menu BA (Berita Acara) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Membangun modul mandiri Menu BA (Berita Acara) untuk ULTG Bekasi yang mencakup Upload BA, Monitoring BA (dengan filter bidang HARGI, HARJAR, HARPRO & tanpa status Open/Close), dan Arsip BA dengan PDF viewer inline.

**Architecture:** Modul ini dibuat menyerupai arsitektur LKS (`lks-dom`). Sub-domain `ba-dom` di bawah parent domain `ANOMALI` di `Header.jsx` dan `Sidebar.jsx` diarahkan ke halaman utama `src/pages/Ba.jsx`, yang membungkus 3 sub-komponen: `BaUpload.jsx`, `BaMonitoring.jsx`, dan `BaArchive.jsx`. Data disinkronkan ke `storageService` dengan key `ultg_ba_list`.

**Tech Stack:** React, Lucide-React, LocalStorage/storageService, JavaScript (ES6+), CSS Inline/App.css.

## Global Constraints

- Menggunakan pola UI/UX yang konsisten dengan halaman LKS.
- Menghindari penggunaan status Open/Close pada Monitoring BA.
- Menyediakan filter 3 bidang utama: HARGI, HARJAR, dan HARPRO.
- Menyimpan file & metadata BA ke `ultg_ba_list` di LocalStorage/storageService.

### Task 2: Service Layer & LocalStorage (`storage.js`)

**Files:**
- Modify: `src/services/storage.js`

**Interfaces:**
- Consumes: Data BA dari `BaUpload`, `BaMonitoring`, `BaArchive`
- Produces: `getBaList()`, `saveBaItem(item)`, `deleteBaItem(id)`

- [ ] **Step 1: Add BA storage methods to `src/services/storage.js`**

Tambahkan helper method untuk BA di `storageService`:
```js
getBaList: () => {
  try {
    const data = localStorage.getItem('ultg_ba_list');
    return data ? JSON.parse(data) : [];
  } catch (e) {
    console.error('Error reading ultg_ba_list:', e);
    return [];
  }
},
saveBaItem: (item) => {
  try {
    const list = storageService.getBaList();
    const existingIndex = list.findIndex(b => b.id === item.id);
    if (existingIndex >= 0) {
      list[existingIndex] = { ...list[existingIndex], ...item, updatedAt: new Date().toISOString() };
    } else {
      list.unshift({ ...item, id: item.id || `ba_${Date.now()}`, uploadedAt: new Date().toISOString() });
    }
    localStorage.setItem('ultg_ba_list', JSON.stringify(list));
    return true;
  } catch (e) {
    console.error('Error saving BA item:', e);
    return false;
  }
},
deleteBaItem: (id) => {
  try {
    const list = storageService.getBaList().filter(b => b.id !== id);
    localStorage.setItem('ultg_ba_list', JSON.stringify(list));
    return true;
  } catch (e) {
    console.error('Error deleting BA item:', e);
    return false;
  }
}
```

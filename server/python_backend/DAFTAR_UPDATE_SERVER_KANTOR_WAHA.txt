Langkah update ringkas di server kantor
---------------------------------------
1. Backup folder server VoltKraft.
2. Copy file kode di daftar atas ke path yang sama.
3. Pastikan MPWA lama tidak berjalan jika nomor WA sama.
4. Jalankan:

   powershell -ExecutionPolicy Bypass -File .\SETUP-WAHA-VOLTKRAFT-SERVER.ps1

5. Buka:

   http://localhost:3000/dashboard

6. Scan QR WhatsApp dan tunggu session default WORKING.
7. Jalankan test:

   .\.venv\Scripts\python.exe .\test_waha_whatsapp_notification.py

8. Jika test berhasil, restart server VoltKraft.

Catatan
-------
Script SETUP-WAHA-VOLTKRAFT-SERVER.ps1 membuat API key random jika tidak
diberi parameter -ApiKey. Simpan hasil API key yang tertulis di waha_api_key.txt.

Jika ingin pakai chat ID tertentu:

   powershell -ExecutionPolicy Bypass -File .\SETUP-WAHA-VOLTKRAFT-SERVER.ps1 -ChatId "120363xxxxxxxxxxxx@g.us"
